Generating online books [closed]

up vote 3 down vote favorite
1

I'm researching technology to create online books, a la:

  • Gradle (definitely my favorite)
  • Git book
  • Spring reference docs
  • Hibernate reference docs

Would anyone know what tools wwere used to create any (or all) of the above?

I'm looking for something that:

  • Separates content from presentation
  • Is simple enough to allow my business users to edit the content (eg markdown, ie not LaTeX)
  • Creates basic navigation, eg 1 page vs. separate page HTML, table of contents, 'next' links, etc
  • Easily skinned
  • Can be generated as part of my build process (eg with maven, jenkins, etc)

Given the quality of the documentation above I have to assume such an animal exists..?

Thanks, Roy

While those pages have been generated with DocBook, don't forget Schacon's project git-scribe, which is a bit more generic.

The git-scribe tool is a simple command line toolset to help you use Git, GitHub and Asciidoc to write e-books. This provides tools for setting up the structure, collaborating with co-authors, doing technical and copy-editing, handling translations, taking errata, as well as publishing online, pdf, mobi (Kindle) and epub (iBooks, Nook) versions.

It uses Asciidoc:

It nicely converts to DocBook and can be used with the GitScribe toolchain.
